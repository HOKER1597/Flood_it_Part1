unzip the archive to a folder of your choice.

Open the Ellipse program.

click: File -> Open Projects from -> Open Projects from File System...

In the dialog box that opens, select the "Directory" option.

Click the "Browse" button and select the folder containing your program.

Click the "Finish" button and Eclipse will automatically create a project based on the contents of the selected folder.

Right click on ExamplesFloodIt file.

In the dialog box that opens, select: Run As -> Java Application(or Alt+Shift+X,J).

Congratulations, the project is launched:)
